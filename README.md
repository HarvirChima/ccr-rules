# ccr-rules

A collection of resources for testing and validating **Copilot code-review rules**.

## Contents

| Directory | Description |
|---|---|
| [`sample-app/`](sample-app/) | Python/Flask REST API – the foundation application for testing code-review rules |

## Sample Application

The `sample-app` directory contains a small but realistic Flask REST API (user management + authentication) that covers a broad set of coding patterns commonly inspected by code-review rules.

See [`sample-app/README.md`](sample-app/README.md) for full details on setup, running, and testing.
